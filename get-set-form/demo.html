GSForm 批量操作表单组件

一、API

方法 参数 返回值 描述
$.GSForm.get() 无 对象数组 元素用id识别，其中同名的radio元素指定第一个radio的id即可
$.GSForm.set(data) 对象数组 无 同上
$.GSForm.getTable(tableId) 无 对象数组
$.GSForm.setTable(data,tableId) 对象数组 无
